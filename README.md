# MoodFit

MoodFit is a Flutter application that offers personalized workout routines tailored to users' current moods and energy levels, making fitness more accessible, engaging, and sustainable.

## Features

### Mood-Based Workout Recommendation System

- Intuitive interface for selecting your current mood
- Algorithm that matches moods to appropriate workout types
- Visually appealing workout cards tailored to mood selection
- Breadcrumb navigation showing your path in the app

### Quick Workout Mode

- 5-10 minute workout routines for users with limited time or energy
- Interactive timers and progress indicators for exercise tracking
- Simple, easy-to-follow exercise demonstrations
- Breadcrumb navigation for context awareness

### Progress Tracking

- System for logging completed workouts and associated moods
- Visual relationship between moods and exercise habits
- Color-coded calendars and progress tracking

### User Authentication and Profile Management

- User registration with email verification
- Secure login with password recovery
- Personalized mood-workout mappings over time
- User preferences and workout history

## Technical Details

### Flutter Framework

MoodFit is built using Flutter for cross-platform compatibility on Android and iOS.

### State Management

The app uses the Provider package for efficient state management.

### Error Handling

- User-friendly error messages for authentication failures
- Network connectivity checks and offline mode capabilities
- Graceful handling of unexpected data formats
- Comprehensive error logging system

## Getting Started

### Prerequisites

- Flutter 3.7.0 or higher
- Dart 3.0.0 or higher

### Installation

1. Clone this repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app on your connected device or emulator
